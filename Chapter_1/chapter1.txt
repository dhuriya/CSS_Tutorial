ðŸ”¹ 1. CSS position Property
The position property defines how an element is placed in the document.

Value	Description
static	Default. Normal document flow.
relative	Shift from its normal spot using top, left, etc.
absolute	Positioned relative to the nearest positioned ancestor (not static).
fixed	Stays fixed in place even on scroll (relative to viewport).
sticky	Scrolls with page until it hits a threshold, then sticks in place.
